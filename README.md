<h1>React - Color Shades & Tints Generator</h1>

</div>

An application which generates a series of shades (ie. lightening) and tints (ie. darkening) in 10% increments from a base hex color provided by the user. This has been built using HTML5, CSS3 (SCSS), JS and Vite. This repo is currently under development.

<strong>:heavy_check_mark: Features</strong>

- Modern UI
- BEM methodology incorporated
- Locally hosted fonts
- Fully responsive
- Optimised for the web
